# Lumera Finance — Sprint 2

Sprint 2 upgrades the Transactions module without changing the existing Supabase schema.

## Included
- Edit and delete with confirmation
- Search by description or category
- Type, category, and month filters
- Five sort modes
- Filtered income, expense, net, and average statistics
- Running balance for the visible ledger
- CSV export
- Responsive desktop/mobile ledger
- Success and error notifications
- ESLint flat configuration

## Deployment
Upload the included files to the root of the existing GitHub repository and commit to `main`. Vercel deploys automatically.
